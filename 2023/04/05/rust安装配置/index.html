# 环境配置

官网安装:
https://www.rust-lang.org/zh-CN/
注意：安装 rust 可能需要 gcc 等依赖，可能还需要 visual studio 安装一些必需模块，按照提示进行安装即可

vscode 插件: rust-analyzer

# Hello World

创建项目

cargo new hello

编写程序主文件: hello/src/main.rs

fn main() {
    println!("Hello, world!");
}

执行代码:

# 进入目录
cd hello/src
#　编译文件
rustc main.rs
# 执行
main

推荐 vscode 插件 Code Runner ，一键编译并执行
